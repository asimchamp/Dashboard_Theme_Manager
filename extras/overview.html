Frequently Asked Questions

Is Splunk Theme Manager compatible with Splunk Cloud?
+
Yes! Splunk Theme Manager works with both Splunk Enterprise and Splunk Cloud (Classic Dashboards).
Can I create my own custom themes?
+
Absolutely. You can use our built-in CSS editor or upload your own CSS files to create fully custom themes.
Does this affect dashboard performance?
+
Negligible impact. We use lightweight CSS injection which is handled natively by the browser's rendering engine.
Is it free to use?
+
Splunk Theme Manager is open-source and free for the community. Enterprise support options are available.
